Обери потрібне

Для Oфісу

Вода необхідна для загальної продуктивності. Коли ви бажаєте, щоб ваша команда була живою кожного дня, почніть із доставки свіжої води. Результати вас здивують —як всіх наших клієнтів.

Для Дому

Здивовані як покращилося здоров'я вашої родини? Все просто —тільки чиста джерельна вода. Можливо це виглядає як маленька зміна, але результат гігантський. Чиста вода покращую метаболізм, прискорює рост дітей, додає сили всьому тілу.


Продукти

Бутильована вода

Замовляй 6л чи 3л, отримаєш + 1л джерельної води.

Вода в бутилках

Також отримаєш знижку, якщо закажеш більше джерельної води.

Вода для кави

Кава найкраща, коли вода в ній відповідна. Якщо хочеш по-справжньому насолодитися кавою, замовляй джерельну воду.

Ваш власний бренд

Отримай воду із назвою твого бренду- розміщай для впізнаваності у конференц залах чи на реценшені.

Про компанію

Чиста сльоза почала свій шлях, як бізнес доствки джерельної води коли засновник фірми Олексій, вирішив доставляти джерельну воду для своєї родини, потім для сімей своїх друзів. Це була маленька сімейна справа по доставці води в будинки по своїй вулиці. Сімейна справа почала рости достатньо швидко і виросла у великий бізнес.
